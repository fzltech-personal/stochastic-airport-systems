This thesis investigates automatic state representation methods for Approximate Dynamic Programming (ADP) applied to stochastic airport gate assignment problems. We formulate the gate assignment problem as a finite-horizon Markov Decision Process (MDP) that explicitly incorporates schedule-based arrivals with stochastic delays and spatially heterogeneous service times constrained by runway-gate topology. We generate state-transition trajectories via simulation to construct a state-transition graph that captures the intrinsic geometry of operational bottlenecks. We study three approaches to basis function construction for linear value function approximation: handcrafted domain features, Proto-Value Functions (PVFs) derived from the spectral decomposition of the state graph, and Successor Features that predict future resource congestion. Our analysis focuses on approximation quality, interpretability, and the ability of learned representations to capture complex spatiotemporal dependencies under varying congestion and disruption scenarios. This work contributes a rigorous comparison of geometric versus engineered representation learning methods in a safety-critical operational domain where interpretability and mathematical transparency are essential.
