This thesis investigates automatic state representation methods for Approximate Dynamic Programming (ADP) applied to stochastic airport gate assignment problems. We formulate the gate assignment problem as a finite-horizon Markov Decision Process (MDP) with stochastic arrivals and service times, then generate state-transition trajectories via simulation under baseline policies. We study three approaches to basis function construction for linear value function approximation: handcrafted domain features, Proto-Value Functions derived from spectral analysis of the state-transition graph, and successor feature representations. Our analysis focuses on approximation quality, interpretability, and scalability under varying congestion and disruption scenarios. This work contributes a rigorous comparison of representation learning methods in a safety-critical operational domain where interpretability and mathematical transparency are essential.
