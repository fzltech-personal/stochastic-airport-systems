PROJECT CONTEXT
---------------
The project studies stochastic airport operations (gate assignment / scheduling under uncertainty)
using Markov Decision Processes (MDPs) and Approximate Dynamic Programming (ADP).

Unlike generic queueing models, this project solves a realistic variant involving:
- **Schedule-Based Arrivals**: Arrivals follow a master schedule with stochastic perturbations (noise), creating structural peaks and valleys in the state space.
- **Heterogeneous Resources**: Service times are spatially constrained by **runway-to-gate topology** (variable taxiing penalties), making some gates strictly more "expensive" than others.
- **Goal**: Compare handcrafted vs. automatically generated basis functions (PVFs, Successor Features) for Linear Value Function Approximation.

This is NOT a deep reinforcement learning project.
Neural networks, if used at all, are limited to representation learning (not control).

DOMAIN
------
Airport-first focus:
- **Gate Assignment** (primary)
- **Runway Constraints**: Planes land on pre-determined runways, creating variable service times based on distance to specific gates.
- **Disruptions**: Stochastic delays, schedule slides, and congestion.

TECHNICAL FOUNDATIONS
---------------------
- **Finite-horizon MDPs**: Explicit state ($t$, $\mathbf{g}$, $\mathbf{q}$), action, and reward definitions.
- **Arrival Process**: Modeled as $A_k = T^{sched}_k + \delta_k$ (Schedule + Noise), not pure Poisson.
- **Graph Construction**: Building **Time-Collapsed** or **Kernel-based** state-transition graphs to avoid "Directed Acyclic Graph (DAG)" pitfalls and ensure spectral connectivity.
- **Approximate Dynamic Programming**: Linear VFA with Temporal Difference learning.

FEATURE / REPRESENTATION METHODS OF INTEREST
--------------------------------------------
- **Handcrafted features** (baseline domain heuristics)
- **Proto-Value Functions** (Eigenfunctions of the *time-collapsed* graph Laplacian)
- **Successor features** (Predicting future runway/gate usage)
- **State aggregation** via spectral clustering
