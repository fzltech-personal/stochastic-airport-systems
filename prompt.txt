PROJECT CONTEXT
---------------
The project studies stochastic airport operations (gate assignment / scheduling under uncertainty)
using Markov Decision Processes (MDPs) and Approximate Dynamic Programming (ADP).

The goal is to:
- Model airport scheduling problems as explicit MDPs
- Generate state-transition trajectories via simulation (not learning)
- Study automatic state representations / basis functions for ADP
- Compare handcrafted vs automatically generated features
- Maintain interpretability, safety awareness, and mathematical clarity

This is NOT a deep reinforcement learning project.
Neural networks, if used at all, are limited to representation learning (not control).

DOMAIN
------
Airport-first focus:
- Gate assignment (primary)

- Disruptions, stochastic delays, congestion, hard constraints

Hospital scheduling is a conceptual analog but not the main domain.

TECHNICAL FOUNDATIONS
